# Navigator 项目概述

## 项目简介
Navigator 是一个现代化的书签管理应用，提供智能书签管理、分类组织和主页网格布局功能。

## 核心功能
1. **书签管理**: 添加、编辑、删除、搜索书签
2. **智能分类**: 支持嵌套分类、拖拽排序、分类置顶
3. **主页网格布局**: 自定义网格列数、书签置顶展示
4. **AI 智能增强**:
   - 自动获取网页元数据（标题、描述）
   - AI 推荐书签分类
   - 智能描述生成
5. **搜索功能**: 基于 Fuse.js 的模糊搜索
6. **导入导出**: 支持书签数据的导入导出
7. **主题定制**: 亮色/暗色主题切换
8. **可调整侧边栏**: 支持宽度调整和收起/展开

## 项目类型
单页应用 (SPA)，使用 Vue 3 + TypeScript 开发

## 开发环境
- macOS (Darwin)
- Node.js + npm
- 开发服务器端口: 3000
