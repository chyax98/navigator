# 扩展图标

## 当前图标

使用了临时占位图标（指南针设计），配色为紫蓝渐变。

## 生成 PNG 图标

需要安装 ImageMagick：

```bash
brew install imagemagick
```

然后运行生成脚本：

```bash
./generate-icons.sh
```

这将从 `icon.svg` 生成以下尺寸的 PNG：
- `icon-16.png` - 16×16（工具栏图标）
- `icon-48.png` - 48×48（扩展管理页面）
- `icon-128.png` - 128×128（Chrome Web Store）

## 自定义图标

如果你有自己的设计：

1. **替换 SVG**：将 `icon.svg` 替换为你的设计（建议 128×128）
2. **运行生成脚本**：`./generate-icons.sh`
3. **或直接提供 PNG**：将 PNG 文件命名为 `icon-16.png`、`icon-48.png`、`icon-128.png`

## 设计建议

- **简洁明了**：在小尺寸下（16×16）也能识别
- **高对比度**：在浅色和深色背景下都清晰可见
- **品牌一致**：与应用主界面配色统一
- **避免文字**：小图标难以显示文字
- **导出格式**：SVG（矢量）+ PNG（位图）

## 配色参考

当前占位图标使用：
- 主色：`#4F46E5`（Indigo 600）
- 渐变色：`#7C3AED`（Violet 600）
- 强调色：`#EF4444`（Red 500，指针）
